# Lumi Desktop Assistant ✨

A modern AI-powered desktop assistant built with:

- React + Vite frontend
- FastAPI backend
- WhatsApp automation
- Email automation
- Smart Excel broadcasting
- Voice assistant support
- Desktop app controls

## Features

- Send WhatsApp messages
- Send emails
- Smart Excel recipient filtering
- Broadcast system
- Modern glassmorphism UI
- Light/Dark mode

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Python
- FastAPI
- PyWhatKit
- Pandas
